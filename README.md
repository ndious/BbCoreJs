## Installation :

Install nodejs and npm

http://nodejs.org/


Install grunt-cli and bower

```
npm install -g grunt-cli bower
```

Install project dependencies
```
npm install --save-dev
```

Download front dependenies
```
bower install
```

## Test and Build the project :
```
grunt
```
